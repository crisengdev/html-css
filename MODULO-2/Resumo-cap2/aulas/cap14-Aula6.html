ITENS NA HORA DE ESCOLHER UMA FONTE NO GOOGLE FONTS

  1. Local aonde deve colar o código gerado pelo site
  2. Nome da Fonte escolhida
  3. Exemplo de como ficará o texto
  4. Aonde deve ser marcado para escolher as variações da fonte
  5. Aba que mostrará o código (Style) e exemplo de código
  6. Local aonde fica o style que deve ser copiado para dentro do código desenvolvido

LOCAL AONDE DEVE SER COLOCADO O LINK IMPORTADO E EXEMPLO DE CÓDIGO NO PROGRAMA
